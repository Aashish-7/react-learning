- Props

## Quick Reference

**Must Learn:**
1. Components & JSX
2. Props & State
3. useEffect
4. Events & Forms
5. React Router
6. Context API

**Learn When Needed:**
- State management libraries
- Form libraries
- Testing
- TypeScript
- Next.js

**Skip For Now:**
- Class components
- Redux (unless required)
- GraphQL (learn REST first)

---

## Learning Tips

1. **Build Projects** - Apply each concept
2. **Practice Daily** - Consistency matters
3. **Read Docs** - Official React docs
4. **Code Along** - Type code yourself
5. **Don't Rush** - Master basics first

---

## Project Ideas by Phase

**Phase 1-2:** Counter, Todo List
**Phase 3-4:** Weather App, Blog Reader
**Phase 5-6:** Task Manager, Shopping Cart
**Phase 7+:** Full CRUD App, Dashboard

---

*Start with Phase 1, complete each step before moving to the next phase.*
